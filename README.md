# bluetti-bt-raw-reader
Tool to find new fields in raw data captured with [bluetti-bt-lib](https://github.com/Patrick762/bluetti-bt-lib)

To run the tool webserver, follow these steps:

```bash
npm i
npm start
```

After that you can open the tool at http://localhost:8080/

## Using without installation

The tool is also available as simple github page at https://patrick762.github.io/bluetti-bt-raw-reader/src/
